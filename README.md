# PredProf

Название проекта: Magical Potions
Описание проекта: Проект представляет собой набор исходных файлов и программу на языке python3. Он позволяет взаимодействовать с таблицей, содержащей информацию о волшебных зельях и их ингредиентах.
Как установить и запустить проект: Проект можно установить, клонировав удаленный репозиторий на свое устройство. Точка входа проекта - файл main.py.
Как использовать проект: В файле main.py определены функции, выполняющие те или иные задачи по поиску, обработке и сохранению данных из таблицы. Их вызов следует производить через функцию main того же файла.
